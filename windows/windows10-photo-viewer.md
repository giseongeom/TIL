# Windows 10의 Photo Viewer

## Windows 10에서 Photo Viewer 사용하기

1. [HKLM_Windows10-PhotoViewer-restore.reg](./HKLM_Windows10-PhotoViewer-restore.reg) 파일 다운로드
2. 관리자 권한으로 `HKLM_Windows10-PhotoViewer-restore.reg` 파일 import
3. `Default apps`-> `Photo viewer`-> `Windows Photo Viewer` 선택
