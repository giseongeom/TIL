## Aptana 환경설정 

### Terminal 설정

* Terminal 에러메시지
```
This Terminal Emulator is not functional because no 'bash' shell could be found
```
* 해결방법
    * Window -> Preferences 선택
    * Team -> Git 선택
    * **Git Executable** 설정에서 git.exe 위치를 지정해 준다. 설정하고 Aptana를 재시작(restart)해야 반영됨
    * Msys git 을 미리 설치해두는 것이 좋다. (Path도 설정되어 있어야 한다)
